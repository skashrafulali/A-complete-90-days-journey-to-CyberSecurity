# 🛡️ Zero to Cybersecurity

A completely free 90-day cybersecurity roadmap for beginners.

This repository documents a structured journey from Linux fundamentals to penetration testing, web security, Python automation, and hands-on labs.

---

## 🎯 Who Is This For?

- Complete beginners
- CSE/IT students
- Future Penetration Testers
- Cybersecurity enthusiasts
- Self-learners

---

## 🧠 Philosophy

> Skills beat certificates.
>
> Consistency beats intensity.
>
> Practice beats passive learning.

This roadmap focuses on building real-world skills using free resources.

---

## 📅 Roadmap Structure

| Phase | Duration | Focus |
|---------|---------|---------|
| Phase 1 | Days 1–30 | Linux, Networking, Python, Web Basics |
| Phase 2 | Days 31–60 | Jr Pentesting, Security Tools |
| Phase 3 | Days 61–90 | HTB, PortSwigger, Bug Bounty |
