🛡️ Zero to Cybersecurity — A Self-Study Roadmap
	A free, practical, day-by-day cybersecurity and Python learning roadmap for absolute beginners. No paid courses. No paywalls. Just consistent work and the right platforms.

👤 Who This Is For
	• Complete beginners with little to no Linux or terminal experience
	• CS/IT students who want to break into penetration testing
	• Anyone who wants a structured path without spending money on courses
	• People who can commit 4–5 hours daily

🧠 Philosophy
	Skills beat certificates. Consistency beats intensity. Free platforms beat paid ones if you use them right.
This roadmap was built on one rule: never hit a paywall. Every resource listed here is 100% free.

🗂️ The Full 90-Day Structure
Phase	Days	Focus
Phase 1	Days 1–30	Linux, Networking, Web Basics, Python Fundamentals
Phase 2	Days 31–60	Jr Pentesting, Python for Security, Real Tools
Phase 3	Days 61–90	HackTheBox Machines, PortSwigger Labs, Bug Bounty Intro
Rest day: Every Friday. A rested brain learns faster.
